Hexo使用笔记

新建文章

hexo new [layout] <title>

[layout]

page source
draft source/_draft

前置申明

title: Hexo使用笔记
tags: Hexo
categories: Hexo
toc: true
description: >-
    记录平时常用的一些Hexo命令.
  • title 文章标题
  • tags 文章标签
  • categories 文章分类
  • toc 是否生成目录
  • description 文章摘要
  • layout 布局
  • date 创建时间
  • updated 修改时间
  • comments 是否开启评论，默认为true
  • permalink 文章永久链接
